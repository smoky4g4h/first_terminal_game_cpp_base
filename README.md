# 🎯 Guess The Number

A simple terminal-based number guessing game written in **C++**.

The player must guess a randomly generated number between **0 and 999**. After each guess, the game provides hints to help the player find the correct number within a limited number of attempts.

## Features

- Random number generation
- Higher/Lower hint system
- Colored terminal output
- Move counter
- Win/Lose conditions
- Sound notifications

## Run Locally

```bash
g++ main.cpp -o game
./game
```

## Concepts Used

- Functions
- Loops
- Conditional Statements
- Random Number Generation
- User Input Handling
